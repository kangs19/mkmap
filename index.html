AI가 분석하는
농산물 가격 위험 신호
기상청 기후 데이터, 농수산물유통공사(KAMIS) 가격·거래량 데이터를 LightGBM 모델로 학습해 14일 후 가격 방향과 급등락 확률을 예측합니다.
🤖
LightGBM 가격 예측 모델
경량 Gradient Boosting 모델로 빠른 학습과 높은 정확도를 동시에 달성합니다. 상승/하락/보합 3클래스 분류 및 급등 확률 산출.
LightGBM · scikit-learn
🗺️
인터랙티브 위험도 지도
전국 시도·시군구 단위 GeoJSON 기반 Leaflet 지도. 생산지 비중, 출하량, 가격 변동, 공급 위험도를 시각화합니다.
Leaflet.js · GeoJSON
⏰
일별 자동 파이프라인
APScheduler 기반으로 매일 오전 6시 KST에 데이터 수집→전처리→모델 추론→DB 저장까지 자동으로 실행됩니다.
APScheduler · SQLite
⚡
FastAPI 백엔드 REST API
비동기 SQLAlchemy ORM + FastAPI로 구축된 RESTful API. 품목별 예측값, 위험 신호, 지역별 데이터를 JSON으로 제공합니다.
FastAPI · SQLAlchemy
☁️
Railway 클라우드 배포
Docker 컨테이너로 패키징해 Railway.app에 배포. 자동 빌드·배포 파이프라인으로 GitHub 푸시 즉시 프로덕션에 반영됩니다.
Docker · Railway · GitHub
📡
